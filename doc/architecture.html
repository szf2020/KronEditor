KronEditor
PLC Integrated Development Environment

KronEditor, IEC 61131-3 standardını hedefleyen masaüstü PLC programlama ortamıdır. Tauri+Vite+React mimarisi ile Linux ve Windows'ta native çalışır; Rust arka ucu ile ST kodu simülasyonu gerçekleştirir.

🦀 Tauri 2 Native Runtime
⚛️ React 18 UI Framework
⚡ Vite 5 Build Tool
🌐 i18n TR / EN / RU
📋 IEC 61131-3 PLC Standard

⚙️ CompilerService.js — ST Kod Üretici

Proje yapısını IEC 61131-3 Structured Text diline dönüştürür; çıktı Tauri üzerinden Rust backend'e gönderilir.

⚠️ Geliştirme Notu
Yalnızca ST tipindeki Program'lar derlenmektedir. Ladder Diagram → ST dönüşümü henüz eklenmemiştir.

📚 LibraryService.js — PLC Kütüphane Yöneticisi

Uygulama başlangıcında /public/libraries/ altındaki XML dosyalarını paralel yükler; kategorilere bölerek Toolbox'a besler.

'/libraries/timers.xml'     // TON, TOF, TP
'/libraries/counters.xml'   // CTU, CTD, CTUD
'/libraries/bit_logic.xml'  // AND, OR, NOT, XOR
06

Data Flow

💾 Kaydetme / Yükleme

  • Kullanıcı "Save" tıklar
    App.jsx → handleSave() tetiklenir
  • XmlService.exportProjectToXml()
    React state → XML string dönüşümü
  • Tauri Plugin FS
    writeTextFile() ile native dosya sistemine yazılır
  • Konsol Logu
    addLog('success', ...) ile kullanıcıya geri bildirim

🚀 Simülasyon

  • Kullanıcı "Simulate" tıklar
    App.jsx → handleSimulate() tetiklenir
  • CompilerService.compileProjectToST()
    Programlar ST formatına dönüştürülür
  • Tauri invoke('simulate_st')
    ST kodu Rust backend'e gönderilir
  • Rust Simülatör
    ST çalıştırılır, sonuç string olarak döner
  • Output Paneli
    Başarı/hata mesajı konsola yazılır
07

PLC Data Model

Proje yapısı IEC 61131-3 organizasyonunu takip eder:

const projectStructure = {
  dataTypes:      [{ id, name, type: 'Array'|'Enumerated'|'Structure', content }],
  functionBlocks: [{ id, name, type: 'ST'|'LD', content: { code, variables } }],
  functions:      [{ id, name, type: 'ST', returnType: 'INT', content }],
  programs:       [{ id, name, type: 'ST'|'LD', content: { code|rungs, variables } }],
  resources:      [{ id, name, type: 'RESOURCE_EDITOR',
                     content: { globalVars, tasks, instances } }]
};

Build & Run

# Geliştirme (Tauri + Vite HMR)
npm run dev

# Sadece web arayüzü
npm run web

# Linux native build
npm run build:linux

# Windows cross-compile
npm run build:windows
ℹ️ Gereksinimler
Tauri 2 için Rust toolchain gereklidir: rustup + cargo. Windows cross-compile için mingw64.
